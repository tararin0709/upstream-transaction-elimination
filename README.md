# 連結会計 未実現利益消去シミュレーター

**連結会計の核心「上流取引未実現利益消去」を、金融システムレベルのJava実装で体現。**

全経上級の会計専門知識を土台に、
税効果（**繰延税金資産**）と**非支配株主持分**までを含む完全な連結修正仕訳をシミュレート。
浮動小数点誤差を排した `BigDecimal` による厳密計算、決算ごとの不変性を担保する `final` 設計、
仕訳をカプセル化したオブジェクト指向構造により、実務で要求される「精度」と「保守性」をコードに刻み込みました。

これは、会計ロジックを堅牢なコードに落とし込む情熱の証です。
次のステップは、圧縮記帳や外貨換算など、さらなる論点へと実装を拡張していくことです。
